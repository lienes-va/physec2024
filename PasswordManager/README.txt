This is a demo project "Password Manager"

For using Fernet you should install cryptography library.
Like this:
pip install cryptography

This is a simple program to save system, username and a password.
System and username should be unique.

Button "Add Password"
User can input service, username and password, which is then encrypted and stored in a local SQLite database.

Button "Retrieve Password"
User can input service name and username to retrieve the associated password. The password will be decrypted and displayed.

Button "Show All Saved Passwords"
Displays all stored passwords in a separate window, where the user can optionally decrypt the passwords (button "Show/Hide).
